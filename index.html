Image Processing - Joint Bilateral Filter

This project is coded in Python and uses the OpenCV library. Joint bilateral filter is an extension of the bilateral filter. The input of the joint bilateral filter is a pair of images, typically a no-flash and a flash image captured in quick succession by a digital camera. Then, a bilateral filter is applied to the first image, using non-spatial masks computed from the second image.
